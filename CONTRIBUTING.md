# Contributing to TRAXR-SOLANA

Thanks for your interest in TRAXR-SOLANA. This repository is an alpha experiment
focused on deterministic pool risk signals for Solana.

## Scope
- Read-only analytics
- JSON fixtures and minimal adapters
- Deterministic scoring and warnings
- No trading, no custody, no signing

## How to Contribute
1. Keep changes small and reviewable.
2. Prefer placeholder heuristics over complex logic.
3. Document TODOs when data is missing.
4. Avoid adding new network dependencies without discussion.

## Pull Requests
- Include a concise description of the change.
- Note any new data fields or API changes.
- If you add fixtures, keep them small and deterministic.

## Reporting Issues
Use GitHub issues to report bugs or request clarifications.
